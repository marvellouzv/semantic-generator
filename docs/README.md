# Материалы для студентов: запуск и разбор Semantic Generator (актуально)

Эта папка — “точка входа” для студентов, которые **не программируют** и впервые запускают репозиторий локально.

### Важно: что сейчас в проекте

- **Модель**: используется **`gpt-5`** (плавающий алиас на latest), через **Responses API** (`AsyncOpenAI.responses.create()`).
- **Где меняются промпты**:
  - `prompts/gpt5_head_queries_fixed_rules.md` — фиксированные правила для этапа генерации кластеров (можно править вручную).
  - `prompts/system_step2.md` — системный промпт для расширения запросов на **Этапе 2** (блок “Расширение запросов”).
  - Динамическая часть (подстановки из UI: тема, выбранные типы, минус-слова, регионы) формируется в коде, например `backend/app/gpt5_head_queries.py`, `backend/app/llm_stage2.py`.

### Ключевые изменения интерфейса (чтобы не путаться)

- **Hotkeys/“Справка”**: отключены (кнопки нет).
- **AI Insights**: удалено.
- **Массовые операции**: удалены (осталась только “Массовая загрузка”).
- **Минус-слова** и **Регионы (для локальных)**: добавлены в Этап 1.
- **Intent vs Group**:
  - **Intent (GPT)**: `commercial | informational | unknown`
  - **Group**: выбранные типы запросов из UI (Коммерческие/Информационные/Сервисные/Ценовые/Локальные/…).

### Что читать / использовать

- `Инструкция_по_первому_запуску.md` — пошаговый запуск (Windows 10/11 + Cursor 2.0).
- `Промпт_для_запуска_Semantic_Generator.md` — готовый промпт, который можно вставить в чат Cursor, чтобы он помог поднять сервис.
- `VIDEO_TELEPROMPTER_STUDENTS.md` — суфлёр (текст “в кадр”).
- `Сценарий_разборного_видео.md` — структура/тайминги и план разборного видео.
- `10_промптов_для_студентов.md` — 10 готовых промптов, как просить Cursor улучшать сервис и чинить ошибки.
